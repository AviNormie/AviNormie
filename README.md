# 🔥 Featured Projects

## 📈 StockFlow
Distributed real-time stock market infrastructure platform built using Go, Kafka, Redis, WebSockets, and Kubernetes.

### Highlights
- Real-time market data ingestion pipelines
- Event-driven microservices architecture
- Low-latency WebSocket streaming
- Redis caching & asynchronous Kafka workflows
- Observability using Prometheus, Grafana & OpenTelemetry

🔗 Live: https://stock-flow-liard.vercel.app/  
🔗 API Docs: https://stock-flow-liard.vercel.app/api/docs

---

## 🌾 Vanik
AI-powered agriculture assistance platform with multi-agent RAG architecture and multilingual voice workflows.

### Highlights
- Multi-agent orchestration using LangChain
- Vector embeddings + cosine similarity search
- Domain-specific AI agents for crop & weather workflows
- Crop disease detection trained on Indian leaf datasets
- Multilingual voice responses using Google TTS
